Melody Maker · 무난작곡기 v4

코드 진행을 넣으면, 규칙(강박=코드톤 · 순차 우선 · 기승전결 단일 클라이맥스)을 지키는 무난하고 조화로운 멜로디 후보를 만들어 ABC 악보로 출력합니다.

16분음표는 기·승·전·결 전체에 등장하되 빈도가 기승전결 곡선을 따릅니다 — 기·결은 드물게(마디당 최대 1쌍), 전은 풍부하게. 항상 짝으로, 순차진행으로만, 마지막 마디는 제외. 복사된 리듬도 독립적으로 장식되어 "변형 반복"이 됩니다.

「기승전결 한번에」는 위 코드 진행을 반복해 기·승·전·결 각 16마디(총 64마디)를 한 흐름으로 이어붙인 완성곡을 만듭니다. 기(낮고 성김) → 승(상승) → 전(클라이맥스·촘촘) → 결(하강·으뜸음 종지)의 곡선을 자동으로 따릅니다.

「완성곡(팝 구성)」은 인트로·벌스·프리코러스·코러스·간주·브릿지·아웃트로를 섹션마다 코드·에너지·리듬·종지를 달리해 한 곡으로 만듭니다(약 60마디). 코러스는 반복해도 같은 후렴, 아웃트로는 으뜸화음으로 착지합니다. 조성(장조/단조)만 고르면 코드 진행은 자동입니다.

「훅(후렴)」은 도입(무난 4마디) → 훅(반복 각인 4·8마디) → 마무리(무난 4마디)를 만듭니다. 훅은 14종 리듬 셀 × 6종 윤곽(정점·하강·상승·진자·반복도약·문답)을 훅 통계 프로파일로 뽑아, 짧은 모티프를 반복 + 당김음 + 정점음으로 각인합니다. 시드를 바꾸면 매번 다른 훅이 나옵니다(조성은 위 완성곡 조성 사용).

사용 순서: 모드 선택 → 생성 → ▶로 미리 듣고 → 괜찮은 후보의 ABC 복사 → 뮤직랩에서 블록 조합. 구간 전용 블록은 이어붙일 때 앞 블록의 끝 음역과 뒤 블록의 시작 음역이 비슷한 후보끼리 고르면 이음새가 매끄럽습니다. 재생은 인터넷 연결이 필요하며(사운드폰트 로드), 재생이 안 되더라도 ABC 복사는 항상 동작합니다.
