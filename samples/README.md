# 테스트용 샘플 (모두 퍼블릭 도메인)

랩 도구들을 시험해 볼 수 있는 샘플 파일입니다. **저작권이 만료된 전통·고전 멜로디**만 담았습니다.
(BitMidi·MIDIWorld 등의 대중가요 편곡은 저작권 문제로 넣지 않았습니다. 그런 곡은 각 사이트에서 직접 받아 쓰세요.)

## 파일

| 파일 | 곡 (출처) | 어떤 도구로 테스트 |
|------|-----------|--------------------|
| `twinkle.mid` / `.abc` | 반짝반짝 작은별 (전통) | ABC↔MIDI 변환기, **분리기(.mid 입력)** |
| `ode-to-joy.mid` / `.abc` | 환희의 송가 — 베토벤 1824 (PD) | ABC↔MIDI 변환기, 분리기 |
| `greensleeves.mid` / `.abc` | 그린슬리브스 (전통, 16세기) | ABC↔MIDI 변환기(단조·임시표) |
| `twinkle-hum.wav` | 반짝반짝 사인파 합성(약 10초) | **오디오→MIDI 변환기** |

## 써보기

- **오디오→MIDI 변환기(audio.html)**: `twinkle-hum.wav`를 넣고 「분석」 → `C C G G | A A G …`(반짝반짝)이 악보로 나옵니다.
- **분리기(split.html)**: `twinkle.mid`를 넣으면 자동으로 ABC로 바꿔 리듬/멜로디로 나눕니다.
- **ABC↔MIDI 변환기(midi.html)**: `.mid`를 드래그하면 ABC로, `.abc`를 붙여넣으면 `.mid`로.

## 출처/라이선스
모두 저작권 만료(퍼블릭 도메인) 멜로디를 랩의 자체 엔진(AbcMidi)으로 생성했습니다. 자유롭게 사용·수정·배포할 수 있습니다.
